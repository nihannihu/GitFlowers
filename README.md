# GitFlowers
A web application for creating and sharing customizable animated flowers with personalized messages.

## Overview
GitFlowers is a lightweight web tool that allows users to create shareable links containing animated flowers with custom messages. The application generates unique URLs that encode the flower configuration, enabling recipients to view fullscreen animated floral displays.

## Features

* **Multiple Flower Types**: Choose from Rose, Tulip, or Lily varieties
* **Custom Color Selection**: Full color picker support for flower customization
* **Personalized Messages**: Add custom text (up to 100 characters) with typewriter animation
* **URL-Based Sharing**: All configurations encoded in shareable links
* **Interactive Elements**: Plant additional flowers by clicking seed elements (maximum 5)
* **CSS Animations**: Grass movement, particle effects, and flower blooming sequences
* **Responsive Design**: Mobile and desktop compatible interface

## Technical Implementation
* HTML/CSS/JavaScript with React via CDN
* No build process or dependencies required
* CSS keyframe animations for visual effects
* Custom web fonts for typography
* Cross-browser clipboard API with fallback support

## Usage

1. Open `index.html` in a web browser
2. Select flower type from available options
3. Choose desired color using the color picker
4. Enter message text in the input field
5. Click "Generate" to create shareable link
6. Copy and share the generated URL
7. Recipients view fullscreen flower display with message
